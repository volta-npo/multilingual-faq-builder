# Multilingual FAQ Builder v1.0 Product Specification

## Workbench

Multilingual FAQ Publisher

## Domain table

FAQ translations

## Metrics

- Translation Coverage
- Review Safety
- Glossary Consistency

## Domain rows

- Source FAQs entered
- Glossary created
- Language tags assigned
- Translations drafted
- Uncertainty flags reviewed
- Native/owner review captured
- HTML/Markdown generated
- CSV exported

## Generated deliverables

- Bilingual HTML
- Markdown FAQ
- Translation CSV

## Validation gates

- Language tags valid
- Source linkage required
- Uncertain translations blocked from approved export
